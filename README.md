# FitTracker

FitTracker is a weekly goal tracking web application created for university.

---

## Installation of application

1. `npm i` to install the required npm packages.
2. `npm run build` to launch the application

Example:

```sh
npm i
npm run build
```

## Starting the application

1. Goto `localhost:8080` in your browser
